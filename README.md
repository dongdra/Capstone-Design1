# 대전 버스 쓰레기통 앱

## 1. 프로젝트 개발 동기
쓰레기를 무단으로 투기하는 상황이 발생을 하게 되어 환겨오염이 초려가 되고 있다. 그래서 이방안을 해결을 하고자 버스 쓰레기통 위치를 찾아서 쓰레기를 버림으로써 환경 오염 방지를 한다.

## 2. 프로젝트 기능
1. 회원가입
2. 로그인, 로그아웃
3. 버스 쓰레기통 위치 알림, 찾기
4. 버스 쓰레기통 위치 추천
5. 공지사항
6. 버스 쓰레기통 마일리지

## 3. 언어, 프레임워크 및 데이터베이스
- **개발 환경**: Windows 11
- **개발 도구**: Visual Studio Code, GitHub, Node.js
- **개발 언어**: JavaScript
- **프레임워크**: React Native
- **데이터베이스**: MySQL
